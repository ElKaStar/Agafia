✅ БОТ УЖЕ ГОТОВ К РАБОТЕ В ГРУППАХ!

┌──────────────────────────────────────────────────────────────────┐
│ ШАГ 1: НАСТРОЙКА В @BotFather (ОБЯЗАТЕЛЬНО!)                     │
└──────────────────────────────────────────────────────────────────┘

1. Откройте @BotFather в Telegram

2. Напишите: /mybots

3. Выберите вашего бота

4. Нажмите: Bot Settings

5. Нажмите: Group Privacy

6. Нажмите: Turn OFF
   (должно стать "DISABLED")
   
   ⚠️  Это критически важно! Без этого бот не увидит сообщения!

7. Вернитесь в Bot Settings

8. Нажмите: Allow Groups?

9. Нажмите: Enable

✅ Готово! Теперь можно добавлять в группу.

┌──────────────────────────────────────────────────────────────────┐
│ ШАГ 2: ДОБАВЛЕНИЕ БОТА В ГРУППУ                                  │
└──────────────────────────────────────────────────────────────────┘

1. Откройте нужную группу

2. Нажмите на название группы сверху

3. Нажмите: Add members (Добавить участников)

4. Найдите вашего бота и добавьте

5. (Опционально) Сделайте бота администратором:
   - Нажмите на бота в списке участников
   - Edit Admin Rights
   - Включите нужные права

┌──────────────────────────────────────────────────────────────────┐
│ ШАГ 3: КАК ОБЩАТЬСЯ С БОТОМ В ГРУППЕ                            │
└──────────────────────────────────────────────────────────────────┘

Бот отвечает в 3 случаях:

1️⃣  УПОМИНАНИЕ ИМЕНИ:

   ✓ "Агафья Петровна, расскажи анекдот"
   ✓ "Агафья, какая погода сегодня?"
   ✓ "агафья петровна дай совет"
   
   Регистр не важен! Можно писать маленькими буквами.

2️⃣  ОТВЕТ НА СООБЩЕНИЕ БОТА:

   ✓ Нажмите "Reply" (Ответить) на любое сообщение бота
   ✓ Напишите свой вопрос
   ✓ Упоминать имя НЕ нужно

3️⃣  УПОМИНАНИЕ ЧЕРЕЗ @:

   ✓ @имя_вашего_бота расскажи историю

┌──────────────────────────────────────────────────────────────────┐
│ ПРИМЕРЫ СООБЩЕНИЙ                                                │
└──────────────────────────────────────────────────────────────────┘

❌ НЕ СРАБОТАЕТ (имя не упомянуто):
   "расскажи рецепт"
   "какая погода?"
   "дай совет"

✅ СРАБОТАЕТ:
   "Агафья Петровна, расскажи рецепт борща"
   "Агафья, какая погода сегодня?"
   "агафья петровна что думаешь о политике?"
   "Привет, Агафья! Как дела?"

┌──────────────────────────────────────────────────────────────────┐
│ В ЛИЧНЫХ СООБЩЕНИЯХ                                              │
└──────────────────────────────────────────────────────────────────┘

В личке бот отвечает на ВСЕ сообщения:

✓ "Привет!"
✓ "Расскажи историю"
✓ "Какая погода?"

Упоминать имя НЕ нужно!

┌──────────────────────────────────────────────────────────────────┐
│ РЕШЕНИЕ ПРОБЛЕМ                                                  │
└──────────────────────────────────────────────────────────────────┘

🔴 БОТ НЕ ОТВЕЧАЕТ В ГРУППЕ:

   Проверьте:
   1. ✅ Privacy Mode ВЫКЛЮЧЕН (DISABLED) в BotFather
   2. ✅ Бот добавлен в группу
   3. ✅ Бот запущен на вашем компьютере (python bot.py)
   4. ✅ В сообщении есть слово "Агафья"

🔴 БОТ ВИДИТ НЕ ВСЕ СООБЩЕНИЯ:

   → Это нормально! Так и должно быть.
   → Бот намеренно игнорирует сообщения без своего имени.
   → Просто упоминайте "Агафья" или "Агафья Петровна"

🔴 БОТ ОТВЕЧАЕТ МЕДЛЕННО:

   → Это нормально, GigaChat думает 3-10 секунд
   → Это скорость работы нейросети
